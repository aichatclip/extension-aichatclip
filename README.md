# @aichatclip/extension

AI チャットサービスからコンテンツをクリップするブラウザ拡張機能。

## ビルド

```bash
# Chrome + Firefox 両方ビルド（デフォルト）
pnpm build

# Chrome のみ
pnpm build:chrome

# Firefox のみ
pnpm build:firefox
```

出力先:
- Chrome: `.output/chrome-mv3/`
- Firefox: `.output/firefox-mv2/`

## 開発

```bash
# Chrome で開発サーバー起動
pnpm dev

# Firefox で開発サーバー起動
pnpm dev:firefox
```
